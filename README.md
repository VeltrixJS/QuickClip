# QuickClip 

Instant replay pour ton écran — sauvegarde les dernières secondes en un raccourci clavier.

---

##  Téléchargement

 **[Télécharger la dernière version (.exe)](https://github.com/VeltrixJS/QuickClip/releases/latest)**

Extrait le `.zip` et lance directement le `.exe` — aucune installation requise.

---

##  Raccourcis

Maintiens **C** + appuie sur des chiffres pour accumuler la durée. Relâche **C** pour sauvegarder.

| Raccourci | Durée |
|-----------|-------|
| `C + 1` | 10 sec |
| `C + 5` | 50 sec |
| `C + 9` | 90 sec |
| `C + 1 + 1 + 1` | 30 sec |
| `C + 9 + 9 + 9` | 270 sec |

> Limite max : **10 minutes**

---

##  Options

- Capture l'écran **principal**, **secondaire**, ou **le dernier écran où était ta souris**
- Tourne en **arrière-plan** dans la barre système
- Clips sauvegardés dans `Vidéos\QuickClips\`
